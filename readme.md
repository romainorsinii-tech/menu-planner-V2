# Planificateur de Menus Hebdomadaires

Une application web simple pour planifier vos repas de la semaine à partir de vos plats favoris.

## Fonctionnalités

- 📝 Ajout de plats favoris avec catégorisation (saison et type)
- 🗑️ Suppression de plats
- 🎯 Génération automatique d'un planning hebdomadaire basé sur vos préférences
- 🔄 Possibilité de "reroll" repas par repas
- 💾 Sauvegarde automatique dans le localStorage du navigateur
- 📱 Design responsive

## Comment utiliser

1. **Ajoutez vos plats favoris** : Remplissez le formulaire avec le nom du plat, sa saison et son type
2. **Définissez vos préférences** : Choisissez la saison et le type de plat souhaité pour la semaine
3. **Générez le planning** : Cliquez sur "Générer le planning"
4. **Ajustez** : Si un repas ne vous convient pas, cliquez sur "Reroll" pour en obtenir un nouveau

## Installation

1. Clonez ce dépôt
2. Ouvrez `index.html` dans votre navigateur
3. Commencez à planifier vos repas !

## Technologies utilisées

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage pour la persistance des données

## Déploiement

Pour déployer cette application sur le web :

1. **GitHub Pages** :
   - Créez un repository GitHub
   - Poussez les fichiers
   - Activez GitHub Pages dans les settings

2. **Netlify** :
   - Créez un compte Netlify
   - Glissez-déposez le dossier du projet

3. **Vercel** :
   - Installez Vercel CLI
   - Exécutez `vercel` dans le dossier du projet

## Personnalisation

Vous pouvez facilement personnaliser l'application en modifiant :
- Les types de plats dans le fichier HTML
- Les couleurs dans le CSS
- Les catégories de saisons
